## **_golang 的安装_**

---

![github star](https://img.shields.io/badge/tattoo1880-v_1.0-blue)

</div>

```shell
wget https://go.dev/dl/go1.24.5.linux-amd64.tar.gz
sudo tar -C /usr/local -xzf go1.24.5.linux-amd64.tar.gz
nano ~/.zshrc
export PATH=$PATH:/usr/local/go/bin
source ~/.zshrc
```
